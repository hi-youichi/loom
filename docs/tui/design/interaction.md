# Loom TUI 交互设计文档

本文档描述 Loom TUI（终端用户界面）的交互方式，不涉及视觉设计和技术实现细节。

---

## 1. 界面布局

### 1.1 整体结构

TUI 界面采用三区域布局，包含顶部标题栏和下方两栏布局：

```
┌─────────────────────────────────────────────────────────────────────┐
│  [标题栏]                    Loom - Agent Dashboard                 │
├────────────────────────────────┬────────────────────────────────────┤
│                                │                                    │
│  [Agent 网格区]                │      [Session 会话区]              │
│  显示所有 Agent 的状态卡片      │      显示当前 Session 的对话       │
│                                │                                    │
│                                │                                    │
├────────────────────────────────┴────────────────────────────────────┤
│  [输入框]                                                           │
│  用于输入任务指令                                                    │
└─────────────────────────────────────────────────────────────────────┘
```

---

### 1.2 各区域详情

#### 标题栏
- 显示应用名称
- 居中对齐
- 青色加粗文字
- 显示 Agent 统计信息

#### Agent 网格区
- 占据左侧 40% 宽度
- 高度自适应（标题栏和输入框之间的所有空间）
- 每个 Agent 显示为独立卡片
- 支持网格布局（多列显示）
- 无 Agent 时显示提示信息
- 可滚动查看更多 Agent

#### Session 会话区
- 占据右侧 60% 宽度
- 高度自适应（与 Agent 网格区相同）
- 顶部显示 Session 选择器
- 显示当前选中 Session 的对话历史
- 包含用户输入和 Agent 回复
- 支持滚动查看历史消息

#### 输入框
- 固定高度 3 行
- 根据模式显示不同提示
- 编辑模式下显示光标

---

## 2. 输入模式

### 2.1 Normal 模式（查看模式）

**状态特征：**
- 输入框边框为灰色
- 输入框文字为灰色
- 无光标显示
- 状态提示：`Input (press 'i' to edit, 'q' to quit)`

**可用操作：**

| 按键 | 功能 |
|------|------|
| `i` | 进入编辑模式 |
| `q` | 退出 TUI |
| `Esc` | 退出 TUI |
| `Ctrl+C` | 强制退出 |
| `Tab` | 在 Agent 网格和 Session 区之间切换焦点 |
| `1-9` | 快速切换到对应编号的 Session |
| `↑/↓` | 滚动当前焦点区域的内容 |
| `Enter` | 查看选中 Agent 的详细信息 |

**适用场景：**
- 查看 Agent 执行状态
- 浏览 Session 会话内容
- 等待任务完成

---

### 2.2 Editing 模式（编辑模式）

**状态特征：**
- 输入框边框为黄色
- 输入框文字为绿色
- 显示光标
- 状态提示：`Input (press 'Enter' to submit, 'Esc' to cancel)`

**可用操作：**

| 按键 | 功能 |
|------|------|
| `Enter` | 提交任务，返回 Normal 模式 |
| `Esc` | 取消输入，返回 Normal 模式 |
| `Backspace` | 删除前一个字符 |
| `任意字符` | 输入到文本框 |

**适用场景：**
- 输入新的任务指令
- 编辑当前输入内容

## 3. Session 管理

### 3.1 Session 选择器

位于 Session 会话区顶部，显示所有可用的 Session：

```
┌─ Sessions ──────────────────────────────────────┐
│ [●] Main     [ ] Code Review  [ ] Testing      │
└─────────────────────────────────────────────────┘
```

**交互方式：**
- 点击 Session 名称切换
- 按数字键 `1-9` 快速切换
- 按 `Tab` 键将焦点移到 Session 选择器
- 使用 `←/→` 方向键在 Session 之间导航

### 3.2 Session 会话内容

显示当前 Session 的对话历史：

**消息类型：**
1. **用户消息**: 用户的输入和指令
2. **Agent 回复**: Agent 的响应和执行结果

**滚动行为：**
- 新消息自动滚动到底部
- 用户手动滚动时暂停自动滚动
- 按 `End` 键跳到最新消息
- 按 `Home` 键跳到最早消息

## 4. Agent 交互

### 4.1 Agent 选择

在 Agent 网格区中：

| 操作 | 方式 |
|------|------|
| 选择 Agent | 点击卡片或使用方向键导航 |
| 查看详情 | 按 `Enter` 或双击 |
| 取消选择 | 按 `Esc` |

### 4.2 Agent 详情视图

选中 Agent 后，在 Session 会话区显示该 Agent 的详细信息：

```
┌─ Agent: CodeAnalyzer ──────────────────────────┐
│ Status: ● Running                              │
│ Task: 重构代码模块                              │
│ Node: reasoning                                │
│ Progress: 正在分析依赖关系...                   │
│                                                │
│ Started: 14:23:45                              │
│ Duration: 00:02:15                             │
│                                                │
│ ─── Execution Log ───                          │
│ [14:23:45] Agent started                       │
│ [14:24:12] Analyzing code structure...         │
│ [14:25:33] Processing dependencies...          │
└────────────────────────────────────────────────┘
```

**可用操作：**
- 按 `Esc` 返回 Session 会话视图
- 按 `r` 刷新 Agent 状态
- 按 `l` 查看完整日志

## 5. 焦点管理

### 5.1 焦点区域

界面有三个可聚焦区域：

1. **Agent 网格区** - 浏览和选择 Agent
2. **Session 会话区** - 查看对话历史
3. **输入框** - 输入新任务

### 5.2 焦点切换

| 操作 | 方式 |
|------|------|
| 下一个区域 | `Tab` |
| 上一个区域 | `Shift+Tab` |
| 直接跳到输入框 | `i` (Normal 模式) |

### 5.3 焦点指示

- 当前焦点区域的边框高亮（白色或黄色）
- 非焦点区域的边框为灰色
- 输入框焦点通过光标显示

---

## 6. 滚动行为

### 6.1 Agent 网格区滚动

- 使用 `↑/↓` 上下滚动
- 使用 `PgUp/PgDn` 翻页
- 使用 `Home/End` 跳到开头/结尾
- 滚动时保持选中状态

### 6.2 Session 会话区滚动

- 使用 `↑/↓` 上下滚动
- 使用 `PgUp/PgDn` 翻页
- 使用 `Home/End` 跳到最早/最新消息
- 新消息自动滚动到底部（除非用户正在查看历史）

## 7. 快捷键总览

### 7.1 全局快捷键

| 按键 | 功能 | 模式 |
|------|------|------|
| `q` | 退出 TUI | Normal |
| `Esc` | 退出 TUI / 取消操作 | Normal |
| `Ctrl+C` | 强制退出 | 任何 |
| `i` | 进入编辑模式 | Normal |
| `Tab` | 切换焦点区域 | Normal |
| `?` | 显示帮助 | Normal |

### 7.2 Agent 网格区快捷键

| 按键 | 功能 |
|------|------|
| `↑/↓/←/→` | 导航 Agent |
| `Enter` | 查看详情 |
| `r` | 刷新状态 |

### 7.3 Session 会话区快捷键

| 按键 | 功能 |
|------|------|
| `1-9` | 切换 Session |
| `←/→` | 在 Session 间导航 |
| `↑/↓` | 滚动消息 |
| `End` | 跳到最新消息 |

### 7.4 输入框快捷键

| 按键 | 功能 | 模式 |
|------|------|------|
| `Enter` | 提交任务 | Editing |
| `Esc` | 取消输入 | Editing |
| `Backspace` | 删除字符 | Editing |

## 8. 帮助系统

### 8.1 帮助覆盖层

按 `?` 键显示帮助覆盖层：

```
┌─ Help ─────────────────────────────────────────┐
│                                                │
│ Global:                                        │
│   q/Esc    - Quit TUI                          │
│   i        - Enter edit mode                   │
│   Tab      - Switch focus area                 │
│   ?        - Show this help                    │
│                                                │
│ Agent Grid:                                    │
│   ↑↓←→    - Navigate agents                    │
│   Enter    - View agent details                │
│                                                │
│ Sessions:                                      │
│   1-9      - Switch session                    │
│   ←/→      - Navigate sessions                 │
│                                                │
│ Press any key to close                        │
└────────────────────────────────────────────────┘
```

### 8.2 状态栏提示

在输入框下方显示上下文相关的快捷键提示：

- Normal 模式: `Input (press 'i' to edit, 'q' to quit)`
- Editing 模式: `Input (press 'Enter' to submit, 'Esc' to cancel)`
- Agent 焦点: `Agent Grid (Enter: details, ↑↓: navigate)`
- Session 焦点: `Sessions (1-9: switch, ↑↓: scroll)`
