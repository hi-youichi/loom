# Session 会话区交互设计

本文档详细描述 Session 会话区的交互方式，包括消息展示、折叠/展开、滚动等。

---

## 1. 消息类型与交互

### 1.1 用户消息 (User Message)

**视觉特征：**
- 蓝色背景 (#0066CC)
- 白色文字
- 前缀图标：👤
- 无折叠功能

**交互：**
- 无特殊交互
- 长文本自动换行

**示例：**
```
┌────────────────────────────────────────┐
│ 👤 You:                                │
│ 请帮我重构 src/lib.rs 文件，优化性能  │
│ 并提高可读性                           │
└────────────────────────────────────────┘
```

---

### 1.2 思考过程 (Thinking)

**视觉特征：**
- 灰色背景 (#3B3B3B)
- 斜体灰色文字
- 前缀图标：💭
- **可折叠/展开**（默认折叠）

**交互：**

| 操作 | 效果 |
|------|------|
| 点击 `▶` | 展开思考过程 |
| 点击 `▼` | 折叠思考过程 |
| `Space` | 切换折叠状态（焦点在该块时） |

**折叠状态：**
```
┌─ 💭 Thinking ──────────────────────────┐
│ ▶ 思考过程 (点击展开)                  │
│    已完成 3 步推理...                  │
└────────────────────────────────────────┘
```

**展开状态：**
```
┌─ 💭 Thinking ──────────────────────────┐
│ ▼ 思考过程                             │
├────────────────────────────────────────┤
│ 1. 分析代码结构...                     │
│    - 发现 5 个函数                     │
│    - 识别 3 处重复代码                 │
│                                        │
│ 2. 评估重构方案...                     │
│    - 提取公共函数                      │
│    - 优化循环逻辑                      │
│                                        │
│ 3. 生成重构代码...                     │
│    - 应用重构模式                      │
│    - 保持向后兼容                      │
└────────────────────────────────────────┘
```

---

### 1.3 工具调用 (Tool Call)

**视觉特征：**
- 黄色背景 (#FFA500，透明度 30%)
- 白色文字
- 前缀图标：🔧
- **可折叠/展开**（默认展开进行中的调用）

**状态：**
- **进行中**: 黄色边框，动画效果
- **成功**: 绿色边框
- **失败**: 红色边框

**交互：**

| 操作 | 效果 |
|------|------|
| 点击 `▶` / `▼` | 折叠/展开参数和结果 |
| `Enter` | 查看详细结果（新窗口或展开） |
| `y` | 复制工具输出到剪贴板 |

**工具调用（进行中）：**
```
┌─ 🔧 Tool: read_file ───────────────────┐
│ ◌ 正在执行...                          │
│ 📄 参数:                               │
│    path: "src/lib.rs"                  │
└────────────────────────────────────────┘
```

**工具调用（成功）：**
```
┌─ 🔧 Tool: read_file ───────────────────┐
│ ✓ 完成 (0.23s)                         │
├────────────────────────────────────────┤
│ ▼ 📄 参数                              │
│    path: "src/lib.rs"                  │
│                                        │
│ ▼ 📤 结果                              │
│    1  fn main() {                      │
│    2      println!("Hello");           │
│    3  }                                │
│    ... (共 150 行)                     │
│                                        │
│ [按 'y' 复制完整输出]                  │
└────────────────────────────────────────┘
```

**工具调用（失败）：**
```
┌─ 🔧 Tool: read_file ───────────────────┐
│ ✗ 失败                                 │
├────────────────────────────────────────┤
│ ▼ 📄 参数                              │
│    path: "src/nonexistent.rs"          │
│                                        │
│ ▼ ❌ 错误                              │
│    Error: 文件不存在                   │
│    Code: FILE_NOT_FOUND                │
└────────────────────────────────────────┘
```

---

### 1.4 助手消息 (Assistant Message)

**视觉特征：**
- 深灰色背景 (#2B2B2B)
- 绿色文字
- 前缀图标：🤖
- 无折叠功能

**交互：**
- 支持文本选择（未来功能）
- `y` 复制消息内容

**示例：**
```
┌────────────────────────────────────────┐
│ 🤖 Assistant:                          │
│ 我已经完成了代码重构。主要改进：       │
│                                        │
│ 1. 提取了重复逻辑到独立函数            │
│ 2. 优化了算法复杂度                    │
│ 3. 添加了必要的注释                    │
│                                        │
│ 重构后的代码性能提升了约 30%。         │
└────────────────────────────────────────┘
```

---

### 1.5 系统消息 (System Message)

**类型：**
- **信息**: 蓝色边框，ℹ️ 图标
- **警告**: 黄色边框，⚠️ 图标
- **错误**: 红色边框，❌ 图标

**视觉特征：**
- 无背景色或浅色背景
- 根据类型使用不同颜色
- 较小的字体或缩进显示

**交互：**
- 无特殊交互
- 自动折叠长错误堆栈（可选）

**示例：**
```
┌─ ℹ️ System ────────────────────────────┐
│ Session created: session-abc123        │
└────────────────────────────────────────┘

┌─ ⚠️ Warning ───────────────────────────┐
│ Token 使用量已达到 80%                 │
└────────────────────────────────────────┘

┌─ ❌ Error ─────────────────────────────┐
│ Agent 执行失败: 超时                   │
│ Details: Operation timed out after 30s │
└────────────────────────────────────────┘
```

## 2. 滚动和导航

### 2.1 自动滚动

**行为：**
- 新消息到达时自动滚动到底部
- 用户手动滚动时暂停自动滚动
- 滚动到底部时恢复自动滚动

**指示器：**
```
┌────────────────────────────────────────┐
│ ▼ 12 条新消息 (按 'j' 跳转)           │
└────────────────────────────────────────┘
```

### 2.2 滚动快捷键

| 快捷键 | 功能 |
|--------|------|
| `j` / `↓` | 向下滚动一行 |
| `k` / `↑` | 向上滚动一行 |
| `Ctrl+D` | 向下滚动半页 |
| `Ctrl+U` | 向上滚动半页 |
| `Ctrl+F` | 向下滚动一页 |
| `Ctrl+B` | 向上滚动一页 |
| `gg` | 滚动到顶部 |
| `G` | 滚动到底部 |

### 2.3 滚动条

**视觉：**
```
│                                        │ █
│  [消息内容]                            │ █
│                                        │ │
│  [消息内容]                            │ │
│                                        │ │ ← 滚动条
│  [消息内容]                            │ │
│                                        │ │
│  [消息内容]                            │ █
```

## 3. 消息过滤和搜索

### 3.1 消息类型过滤

**快捷键：** `F` (Filter)

**过滤选项：**
```
┌─ Filter Messages ──────────────────────┐
│ [✓] 用户消息                           │
│ [✓] 思考过程                           │
│ [✓] 工具调用                           │
│ [✓] 助手消息                           │
│ [✓] 系统消息                           │
│                                        │
│ [Enter] 应用  [Esc] 取消               │
└────────────────────────────────────────┘
```

### 3.2 搜索功能

**快捷键：** `/` (Search)

**搜索栏：**
```
┌─ Search ───────────────────────────────┐
│ 搜索: refactor█                        │
│                                        │
│ 找到 3 个匹配项                        │
│ [Enter] 下一个  [Shift+Enter] 上一个   │
│ [Esc] 关闭                             │
└────────────────────────────────────────┘
```

**搜索导航：**
- `n` - 下一个匹配项
- `N` - 上一个匹配项

## 4. 消息操作

### 4.1 复制消息

**快捷键：** `y` (Yank)

**行为：**
- 焦点在某个消息块时，按 `y` 复制该消息内容
- 复制成功后显示提示：
  ```
  ✓ 已复制到剪贴板
  ```

### 4.2 查看详情

**快捷键：** `Enter`

**行为：**
- 对于工具调用：展开/折叠详细参数和结果
- 对于思考过程：展开/折叠完整推理
- 对于其他消息：无效果

### 4.3 折叠/展开所有

**快捷键：**
- `zc` - 折叠所有可折叠消息
- `zo` - 展开所有可折叠消息
- `za` - 切换所有折叠状态

## 5. 性能优化

### 5.1 虚拟滚动

**策略：**
- 只渲染可见区域的消息
- 缓冲区：上下各保留 5 条消息
- 超出缓冲区的消息从 DOM 中移除

### 5.2 消息限制

**限制：**
- 最多保留 1000 条消息
- 超出时删除最旧的消息
- 可配置（配置文件中设置 `max_messages`）

### 5.3 长消息截断

**策略：**
- 单条消息最大高度：终端高度的 50%
- 超出时显示截断指示器：
  ```
  ... (截断了 500 行，按 Enter 查看完整内容)
  ```

## 6. 焦点管理

### 6.1 焦点指示

**视觉：**
- 焦点消息块左侧显示 `┃` 标记
- 焦点消息块边框加粗

```
┃ ┌─ 🔧 Tool: read_file ─────────────┐
  │ ✓ 完成                           │
  └───────────────────────────────────┘
```

### 6.2 焦点导航

| 快捷键 | 功能 |
|--------|------|
| `J` | 移动到下一条消息 |
| `K` | 移动到上一条消息 |
| `{` | 移动到上一条同类型消息 |
| `}` | 移动到下一条同类型消息 |

## 7. Session 切换

### 7.1 Session 选择器

**位置：** Session 会话区顶部

**交互：**
```
┌─ Sessions ──────────────────────────────┐
│ [●] main-session                        │
│ [ ] code-review                         │
│ [ ] debug-session                       │
└─────────────────────────────────────────┘
```

**快捷键：**
- `s` - 打开 Session 选择器
- `1-9` - 快速切换到第 N 个 Session
- `Tab` - 在 Session 列表中循环

### 7.2 Session 状态

**图标：**
- `●` - 活跃 Session（当前查看）
- `○` - 非活跃 Session
- `◉` - 有新消息的 Session
- `✓` - 已完成的 Session
- `✗` - 出错的 Session

---

## 8. 响应式设计

### 8.1 窄终端 (<100 列)

**调整：**
- Session 会话区占 100% 宽度
- Agent 网格区隐藏
- 使用 `Tab` 切换显示

### 8.2 中等终端 (100-140 列)

**调整：**
- Agent 网格区：30% 宽度
- Session 会话区：70% 宽度

### 8.3 宽终端 (>140 列)

**调整：**
- Agent 网格区：40% 宽度
- Session 会话区：60% 宽度
